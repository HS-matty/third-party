Administration


Welcome! This is the main PHP-Nuke administration screen.

Last XX articles: Show to you the last XX articles published in your news site. XX is the number that you set for $admart in config.php file. You will see as first record the last article published. Making click on the story's name you can edit the story or just use the Edit, Delete links at the right. If you want to edit/delete an older story, just type the Story ID in the form, choose Edit or Delete option and click OK button.
