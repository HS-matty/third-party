NEW Article


This is main function for your site, the possibility to add articles. All what you write here will appear in the Home of your site, so be carefully and double check the URLs and all the info.

Title: Write the story title, try to be short and descriptive.

Topic: Select the desider topic for this story. If no topic is selected the default selection will be the first topic in the database.

The Story: The text that will appear in the Home. You can use HTML code.

Extended Text: This space is to add more info on your story. If your article is a long one, the biggest part will go here. If there are Extented Text you will see a Read More... link in the Home under this story box. You can use HTML code.

PreviewAdminStory: Select this option to see a preview of this story. It's a good practice to make a story review and check for gramma and URLs. This option will open another page with the story preview, read it and correct the text if needed, then you can post the story.

PostAdminStory: If you are 100% sure that your story is well written and the URLs are OK you can Post it from here.
