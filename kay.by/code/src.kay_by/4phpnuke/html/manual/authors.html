Edit Admins


In this section you can add, modify and delete administrators for your site.

To add a new Admin fill the form. The required fields are marked with * like Nickname, Name, Email and Password. The othe field, URL, is optional. When you finished to fill the info just click on Add Author button to add the new administrator.

To edit an existing Admin just click on Modify Info on the selected Admin. This will open another page with that Admin's data, just change what you need and click on Update Author button. If you need to change the Admin's password, just type twite the new password.

To delete an existing Admin click on Delete Author for the selected Admin.

NOTE: The Admin account with the name God can't be deleted, this because PHP-Nuke needs at least one administrator.
